RxMemo 구현해보기
=============
#### RxSwift + MVVM, 화면전환을 위한 Coordinator를 이용하여, RxMemo 앱을 구현하였습니다.
   
## 적용된 버전
### XCode 13.2, Swift 5
   
## 사용한 라이브러리
* RxSwift
* Action
* NSObject+Rx
* RxDatasources
* RxCoreData

## 트러블 슈팅
### RxDataSources를 평소에 사용하지 않았었고, RxCocoa를 이용해서 화면 구현만 해보았는 데, 적용하는게 그리 어렵진 않았지만, 새롭게 다가와서 공부하는 시간과 서칭하는 시간이 좀 필요하였습니다.
